# Domain Mapping

## /apps/portfolio

- `src/content/`: Source of truth for Project and Blog data.
- `src/components/ui/`: Stateless, declarative Astro components.
- `src/components/islands/`: Interactive Svelte 5 components.
- `src/lib/`: Pure functional logic (math, string parsing, API clients).

## /docs

- Contains the "Global Knowledge Base" for AI agents.
- `mock.md` serves as the visual and structural baseline.
